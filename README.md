# Adwaita Extended

This is an extension pack for GNOME's Adwaita icon theme based on [GNOME Human Interface Guidelines](https://developer.gnome.org/hig/).

## Supported

* 010 Editor
* Arduino IDE
* Burp Suite Professional
* Genymotion
* GParted
* Icalingua
* Inkscape
* MongoDB Compass
* NetEase Cloud Music Gtk
* OBS Studio
* Oracle VM VirtualBox (OSS)
* Quartus Prime Lite
* Steam
* Sublime Text
* TeamViewer
* V2rayA
* Ventoy
* Visual Studio Code
* WeChat
* WemeetApp
* Wireshark
